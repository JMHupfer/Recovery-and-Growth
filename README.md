# Jacob M Hupfer — Portfolio (Developer)

This repository contains a developer-focused portfolio for Jacob M Hupfer. The site highlights coding work, projects, and technical writing, and uses a darker "galaxy" visual theme.

## What’s included
- Landing page with a developer-focused hero and featured projects
- Pages for Resume, Writing (technical posts), and Contact
- Responsive sidebar navigation with a mobile hamburger menu
- Contact form (FormSubmit) for incoming messages
- A lightweight reveal-on-scroll animation and accessible layout

## Live site
https://jmhupfer.github.io/Jakes-First-Portfolio/

## Development
To preview locally, run a simple static server from the project root (Python 3):

```bash
python -m http.server 8000
# then open http://localhost:8000
```
